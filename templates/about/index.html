Idealab

Idealab is a forum for new open-source project ideas. If you would like to submit an idea, please fill out the following form. Once the idea is approved it will appear on the IdeaLab webpage. Once ideas are approved and migrated, your idea will officially be a project and will appear in the Projects webpage.

Projects

All Projects visible on the MakeOpenSource Github page are open to contributions. Each project will have a contributions.txt file to specify a contribution format unique to that project, to ensure that submission adhere to existing formats.
